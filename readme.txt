для создания виртуального окружения воспльзуйтесь командой командной строки:
python -m venv venv
после чего необходимо активировать виртуальное окружение:
cd venv\Scripts && activate
после чего можно запускать скрипт, например так:
python ../../main.py C:\Windows\System32 C:\masm64\lib C:\masm64\inc
